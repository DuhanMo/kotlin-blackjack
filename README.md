### 기능 구현사항
- [x] 카드는 랭크와 문양으로 이루어진다.
- [x] 카드의 랭크는 2~10,J,Q,K,A 이다.
- [x] 랭크가 숫자이면 점수는 숫자의 값으로 계산된다.
- [x] 랭크가 J,Q,K 이면 점수는 10점으로 계산된다.
- [x] 랭크가 에이스이면 기본점수는 11점이다.
- [ ] 카드들의 점수합이 21이 넘을 경우 에이스는 1점으로 보정된다.
- [ ] 카드목록의 점수는 카드의 점수의 합이다.
- [ ] 카드를 섞은 덱을 만든다.
- [ ] 유저는 카드 목록을 보유한다.
- [ ] 카드목록에서 점수를 계산한다.
- [ ] 카드를 덱에서 하나 뽑는다.
