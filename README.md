### 기능 구현사항
- [x] 카드는 랭크와 문양으로 이루어진다.
- [x] 카드의 랭크는 2~10,J,Q,K,A 이다.
- [x] 랭크가 숫자이면 점수는 숫자의 값으로 계산된다.
- [ ] 랭크가 J,Q,K 이면 점수는 10점으로 계산된다.
- [ ] 랭크가 에이스이면 점수는 1 또는 11점으로 계산되며 상황에 따라 유리한 점수를 선택한다.
- [ ] 카드를 섞은 덱을 만든다.
- [ ] 카드를 덱에서 하나 뽑는다.
